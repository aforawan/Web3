1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

• CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan visual (seperti warna, font, tata letak) pada halaman web, sementara JavaScript digunakan untuk menambahkan interaktivitas dan fungsi dinamis (seperti animasi, validasi formulir, dll.) pada halaman web.

2.Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web

• CSS (Cascading Style Sheets) dan JavaScript dapat diintegrasikan dalam pemrograman web untuk mengontrol tampilan dan interaktivitas halaman web.

3.Jelaskan apa saja keuntungan menggunakan JavaScipt dan CSS?

• Menggunakan JavaScript dan CSS memberikan banyak keuntungan dalam pengembangan web, termasuk kemampuan untuk membuat tampilan yang lebih menarik dan interaktif, serta meningkatkan efisiensi pengembangan.

4 Jelaskan apa saja keuntungan menggunakan Bootstrap?

• Melalui Bootstrap, pengguna dapat mengembangkan website responsif dengan mudah tanpa perlu melakukan coding dari awal. Sebelum menggunakan Bootstrap, jangan lupa untuk memastikan website kamu memiliki performa yang baik dan dapat diakses dengan cepat.
